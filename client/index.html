    Recommend

    이 프로그램을 쓰는 분들께 추천합니다

    • Visual studio code

      기초가 단단해야 높은 곳까지 오를 수 있습니다. 공간 감각과 형태 이해부터, 3D 세계로 향하는 탄탄한 출발을 지원합니다

    • Figma

      자연스럽고 몰입감 있는 3D 렌더링을 위해 조명과 색채 이론을 깊이 있게 다루어, 보는 이의 시선을 사로잡는 작품을 완성합니다

    • Docker

      현실감 넘치는 모델링을 위해 필수적인 디테일 작업을 배우고, 하이퀄리티 3D 모델을 완성하는 실전 기술을 마스터하세요

    • PyCharm

      브랜드 아이덴티티를 녹여낸 3D 아트워크를 통해, 세상에 단 하나뿐인 당신만의 독창적인 작품 세계를 구축합니다

    • Jupyter Notebook

      리얼타임 엔진과 하이엔드 렌더러를 자유자재로 다루며, 차별화된 기술력으로 3D 표현의 극한을 경험해보세요

    • Google Colab

      리얼타임 엔진과 하이엔드 렌더러를 자유자재로 다루며, 차별화된 기술력으로 3D 표현의 극한을 경험해보세요
